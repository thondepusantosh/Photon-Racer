# Description

A fast-paced futuristic bike racing and shooting game where players race through high-tech tracks while battling enemy bots. Speed, accuracy, and strategy are key to survival and high scores.

# Gameplay

Players control a futuristic bike, race at high speeds, and shoot hostile enemy bots in real time. Destroying enemies earns points while avoiding obstacles and attacks keeps the race alive.

# Enemy Bots

AI-controlled enemy bots attempt to block the player and attack during the race. Each bot defeated contributes to the player‚Äôs score.

# Combat System

Integrated shooting mechanics allow players to attack enemies while riding. Precision and timing are essential for effective combat.

# Scoring System

Points are awarded for eliminating enemy bots. Higher accuracy and faster gameplay lead to better scores.

# Objective

Survive the race, defeat as many enemy bots as possible, and achieve the highest score.

# Environment

Set in a futuristic sci-fi world with advanced bikes, neon-style tracks, and immersive visuals.
